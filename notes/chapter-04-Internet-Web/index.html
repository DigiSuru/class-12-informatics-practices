1. Introduction to Computer Networks

A Computer Network is an interconnection of two or more computers or computing devices to share data and resources among each other.

💻 Node / Host

Any device (computer, printer, router, etc.) connected to a network that can send, receive, or store data.

📦 Packet

Data sent over a network is divided into smaller chunks called packets.

Geographical Network Types

🔥 Exam Tip: Remember the size order: PAN < LAN < MAN < WAN.

📱

PAN

Personal Area Network

  • Very short range (few meters)
  • Phones, Laptops, Bluetooth
  • Easy to use, low speed
🏢

LAN

Local Area Network

  • Up to 1km (Room, Building)
  • 10 Mbps to 1 Gbps
  • Uses Ethernet rules
  • Highly secure
🏙️

MAN

Metropolitan Area Network

  • 30-40km (Town, City)
  • Connects multiple LANs
  • Ex: Cable TV networks
🌍

WAN

Wide Area Network

  • Countries / Continents
  • Connects LANs and MANs
  • The Internet is a WAN

Types of Networks Based on Architecture

🤝 Peer-to-Peer (P2P)

All computers are equal and there is no central server. Each computer can act as both a client and a server. Simple and inexpensive, suitable for small networks.

✓ Cheap ✗ Poor Security

👑 Client-Server

One or more computers act as servers and provide resources/services, while other computers (clients) request those services. Server controls security and access.

✓ Highly Secure & Efficient ✗ High Cost

🔹 Intranet

A private network used within an organization. Allows employees to share information securely.

🔹 Extranet

An extension of an intranet that allows limited access to external users such as clients or business partners.

Internet vs WWW

🌍 The Internet

The global network of computing devices. Not just PCs, includes servers, tablets, phones, and smart IoT devices. Connects via Modem → local ISP → Regional Net → National Net → Internet Backbone.

🕸️ The Web (WWW)

An ocean of information stored as trillions of interlinked web pages. It is a software/service running on top of the Internet. (Invented by Sir Tim Berners-Lee in 1990).

⚡ 3 Fundamental Web Technologies:

  • HTML: Standard language to design web pages using tags.
  • URI/URL: Uniform Resource Identifier (identifies resource). URL is a type of URI that provides the specific location and protocol (e.g., http://www.ncert.nic.in).
  • HTTP/HTTPS: Set of rules to retrieve and transfer web pages. HTTPS is secure/encrypted.

Web Server

A web server stores website contents and delivers them to clients upon request.

  • As Hardware: A powerful physical computer connected to the Internet storing HTML, CSS, JS, and images.
  • As Software: A specialized program accepting HTTP requests and sending HTTP responses. If page is missing, sends Error 404.

Static vs Dynamic Web Pages

Static Web Page (.htm / .html)

Content remains the same for every user.

Processing: Server simply locates the page and sends it. No server-side processing.

Languages: HTML, CSS, JavaScript. Speed: Loads faster.

Hosting of a Website

Web hosting is renting server resources (CPU, RAM, storage) from a Cloud Service Provider. A server has a numeric IP address, but DNS (Domain Name System) maps textual domain names to this IP.

⭐ Steps to Host: Mnemonic "S.R.S.U.M"

Choose provider based on needs (bandwidth, DB, storage).
Identify a suitable name and purchase from a Domain Name Registrar.
Create login credentials and note down the IP address.
Transfer HTML, CSS, and Database files into proper folders.
Link your registered Domain to the server's IP via DNS settings.

Internet Services

📧 Electronic Mail (Email)

Sending/receiving messages. Custom business domains (username@company.com). Key facilities:

  • ✓ Composing & Saving Drafts
  • ✓ Broadcasting to multiple users
  • ✓ CC (Carbon Copy) & BCC
  • ✓ Forwarding mail
  • ✓ Filtering Spam
  • ✓ Folders & Organization
  • ✓ Address Book management
  • ✓ Auto-Signatures (Footers)
  • ✓ Printing/Saving files
  • ✓ Searching old emails

💬 Chat (Instant Messaging)

Real-time communication. Both parties must be online simultaneously using the same app (WhatsApp, Slack, Skype). Supports text, audio, video, and groups.

📞 VoIP (Voice over IP)

Broadband Telephony. Converts analog voice to digital packets over the internet.

Pros: Free/cheap (great for international), high portability using IP phones.
Cons: Quality depends completely on internet speed.

Web Browsers

A software application (Chrome, Firefox, Opera) used to view web pages and retrieve info from servers.

⚙️ Browser Settings (e.g., Firefox)

  • General: Default browser, language, downloads.
  • Home: Set the default home page.
  • Search: Manage default search engines.
  • Privacy & Security: Tracking protection, passwords, history, cookies.
  • Sync: Manage account across devices.

🧩 Add-Ons vs. Plug-ins

Plug-in: A complete, standalone 3rd-party software installed on the computer that the browser uses.
Ex: Java, Flash Player (to play video).
Add-on (Extension): Not a complete program. A small script added directly to the browser for specific functionality.
Ex: Ad-blockers, Grammar checkers.

🍪 Cookies

A small text file containing info transferred by a website to your browser. Helps the server recognize you.

  • Remembers shopping carts, logins, languages.
  • Security: They are harmless plain text. Cannot transmit viruses or read hard disks.
  • Users can clear/disable them via Privacy settings.
